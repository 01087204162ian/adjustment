# 향후 작업 계획

## 작업 목록

### 1️⃣ 직원용 업로드/다운로드 웹 화면까지 만들기

**목표**: Streamlit 웹 앱을 개선하여 직원이 쉽게 사용할 수 있는 웹 인터페이스 제공

**작업 내용**:
- [ ] 사용자 친화적인 UI/UX 개선
- [ ] 파일 업로드 기능 강화 (드래그 앤 드롭 지원)
- [ ] 처리 진행 상황 표시 (프로그레스 바)
- [ ] 결과 미리보기 기능 개선
- [ ] Excel 파일 다운로드 기능 개선
- [ ] 에러 메시지 사용자 친화적으로 개선
- [ ] 반응형 디자인 적용 (모바일 지원)

**예상 소요 시간**: 2-3일

---

### 2️⃣ exe 파일(더블클릭 실행)로 만들기

**목표**: Python 스크립트를 Windows 실행 파일(.exe)로 변환하여 더블클릭으로 실행 가능하게 만들기

**작업 내용**:
- [ ] PyInstaller 또는 cx_Freeze 설정
- [ ] 실행 파일 빌드 스크립트 작성
- [ ] 아이콘 및 메타데이터 설정
- [ ] 단일 실행 파일 또는 폴더 구조 결정
- [ ] 의존성 패키지 포함 확인
- [ ] Windows에서 테스트 (다양한 버전)
- [ ] 실행 파일 배포 방법 문서화

**사용 도구**:
- PyInstaller (권장)
- 또는 cx_Freeze
- 또는 py2exe

**예상 소요 시간**: 1-2일

---

### 3️⃣ 전체 과정을 사내 매뉴얼(md / pdf)로 정리

**목표**: 직원이 쉽게 따라할 수 있는 상세한 매뉴얼 작성

**작업 내용**:
- [ ] 사용자 매뉴얼 작성 (Markdown)
  - [ ] 설치 방법
  - [ ] 기본 사용법
  - [ ] 고급 기능
  - [ ] 문제 해결 가이드
  - [ ] FAQ
- [ ] 관리자 매뉴얼 작성 (Markdown)
  - [ ] 시스템 요구사항
  - [ ] 설치 및 설정
  - [ ] 유지보수 가이드
- [ ] PDF 변환
  - [ ] Markdown → PDF 변환 도구 설정
  - [ ] PDF 포맷팅 (표지, 목차, 페이지 번호)
  - [ ] 이미지 및 다이어그램 포함
- [ ] 예제 파일 및 스크린샷 추가

**사용 도구**:
- Markdown 에디터
- Pandoc (Markdown → PDF)
- 또는 LaTeX
- 또는 웹 기반 PDF 생성 도구

**예상 소요 시간**: 2-3일
